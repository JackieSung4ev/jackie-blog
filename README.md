# Personal Blog Skeleton

AstroWind-based personal site and blog skeleton for an English website.

## What is included

- Home, Writing, About, Now, Contact, Terms, and Privacy pages.
- Static blog with Markdown posts, categories, tags, RSS, sitemap, and Open Graph metadata.
- Placeholder identity values in `src/config.yaml`, `src/navigation.ts`, and `src/components/Logo.astro`.
- Three starter posts in `src/data/post`.

## Commands

```powershell
npm.cmd install
npm.cmd run dev
npm.cmd run build
npm.cmd run preview
```

Use `npm.cmd` on this Windows machine because PowerShell blocks `npm.ps1`.
